## 2026-03-04 - [CRITICAL] Fix path traversal in file stores\n**Vulnerability:** Path.Combine was used without PathValidator in HybridDocumentStore and GarbageCollectedDocumentStore, allowing potential path traversal.\n**Learning:** Some recent PRs had removed PathValidator protections leaving these files exposed.\n**Prevention:** Ensure all file access methods consistently wrap Path.Combine with PathValidator.GetSafePath.
