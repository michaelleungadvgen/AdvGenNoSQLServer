=== AdvGenNoSQL Server Performance Benchmarks ===

Available benchmark suites:
  1. DocumentStoreBenchmarks - CRUD operation performance
  2. QueryEngineBenchmarks - Query parsing and execution performance
  3. BTreeIndexBenchmarks - Index operation performance
  4. CacheBenchmarks - Cache hit/miss and eviction performance
  5. SerializationBenchmarks - JSON and message serialization performance
  6. All - Run all benchmarks
